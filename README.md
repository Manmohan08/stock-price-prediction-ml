# Stock Market Prediction using Machine Learning

This project analyzes historical stock market data using Python and machine learning algorithms.

## Technologies Used:
- Python
- Pandas
- NumPy
- Scikit-learn

## Description:
The model processes historical stock data and identifies patterns to generate predictions for future price trends.

## Author:
Manmohan Kumar

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Manmohan08/stock-price-prediction-ml.git
   cd stock_market_prediction
   ```

2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install the required dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the prediction model:**
   ```bash
   python main.py
   ```
